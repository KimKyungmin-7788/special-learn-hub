## 관리자 기능 구현 계획

### 1. DB 테이블 생성 (마이그레이션)
- `tools` — 에듀테크 도구 (이름, 설명, URL, 카테고리, 대상, 썸네일 등)
- `announcements` — 공지사항 (제목, 내용, 배너 이미지, 활성 여부)
- `user_roles` — 관리자 권한 (admin role)
- RLS 정책: 누구나 읽기 가능, admin만 쓰기/수정/삭제

### 2. 인증 (로그인/회원가입)
- 이메일+비밀번호 로그인/회원가입 페이지
- AuthContext로 세션 관리
- 헤더의 로그인 버튼 연동

### 3. 관리자 페이지 (`/admin`)
- 관리자만 접근 가능
- 탭 구성: 도구 관리 | 공지 관리
- 도구 CRUD (추가/수정/삭제 폼)
- 공지 CRUD (제목, 내용, 배너 이미지 URL)

### 4. 프론트엔드 연동
- 기존 하드코딩된 샘플 데이터 → DB에서 실시간 조회
- 공지 배너도 DB에서 조회
